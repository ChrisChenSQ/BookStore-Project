This project is for the CMPSC431W online book store Project.
This is only a demo version of the website.

Required Environment:
Python 3.9
Django 3.2

If Django is not Installed, use pip to install:
pip install Django

To Run the project, open the entire file as a project in the python console. Then use the commend line  ' python manage.py runserver' to run the project. 

In this Project, the base HTML is in the Book file. In each APP file, there will be a template file store the HTML used in that APP. The functions of the website are all stored inside the views.py in each file.
The models.py stored the information used for Django to communicate with the database.
